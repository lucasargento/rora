## Experiments

Repositorio de experimentos sobre agentes y razonamiento.

### Estructura

- **notebooks (`*.ipynb`)**: ejemplos y análisis rápidos. Empezá por `agent_example.ipynb` y `example.ipynb`.
- **`src/`**: código fuente del agente y utilidades (p. ej., `src/agent/agent.py`, `src/agent/gates/`, `src/agent/nodes/`).
- **`paper/`**: borrador y artefactos del documento (figuras, tablas, bibliografía).
- **`outputs/`**: salidas generadas por corridas de razonamiento y registros asociados.

### Uso rápido

- Abrí un notebook y ejecutá las celdas para reproducir los ejemplos.
- Para usar el código desde `src/`, asegurá que el directorio esté en tu `PYTHONPATH` al ejecutar los notebooks.
